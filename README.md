## Project Description

This app works as a digital cookbook where users can search for recipes from the API and create their own recipes. Users can save and rate recipes they like and easily find them later in a favorites list.

## Installation
Clone the repository: git clone https://github.com/ElliotPa1m/frontendramverk-grupp-1.git
Install dependencies: npm install
Start the project: npm run dev
Open your browser and go to: http://localhost:5173
